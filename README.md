# 📊 Analyzing Students' Mental Health (DataCamp Project)

## 📝 Project Overview
Does staying in a foreign country for university affect your mental health? This project explores a dataset from a Japanese International University (2018) to analyze how the **Length of Stay** impacts the mental health scores of international students.

The study focuses on three key diagnostic tests:
* **PHQ-9** (Depression test)
* **SCS** (Social Connectedness Scale)
* **ASISS** (Acculturative Stress Index)

## 🎯 Objectives
The goal was to extract a summary table showing:
1. The number of international students for each length of stay.
2. The average diagnostic scores (PHQ, SCS, AS) rounded to two decimal places.
3. Results sorted by the length of stay in descending order.

## 🛠️ Skills Demonstrated
* **SQL Aggregations:** Using `AVG()` and `COUNT()` to summarize data.
* **Data Filtering:** Using `WHERE` to isolate international students (`inter_dom = 'Inter'`).
* **Data Formatting:** Using `ROUND()` for professional reporting.
* **Grouping & Sorting:** Mastering `GROUP BY` and `ORDER BY` for structured insights.
